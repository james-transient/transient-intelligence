# Security Policy

## Reporting a Vulnerability

If you discover a security issue, do not open a public issue.

Please report it privately to: **support@transientintelligence.com**

Include:

- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Any suggested mitigation

## Scope

This repository contains public-safe materials only. Do not include secrets, tokens, private infrastructure details, or proprietary internals in reports or pull requests.
